## Spiral Matrix II
[Source](https://leetcode.com/problems/spiral-matrix-ii/)

Given a positive integer n, generate an n x n matrix filled with elements from 1 to n2 in spiral order.

## Constraints:

 - 1 <= n <= 20

## Example 1:
```sh
Input: n = 3
Output: [[1,2,3],[8,9,4],[7,6,5]]
```

## Example 2:
```sh
Input: n = 1
Output: [[1]]
```
