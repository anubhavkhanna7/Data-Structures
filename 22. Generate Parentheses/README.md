# Generate Parentheses
[Source](https://leetcode.com/problems/generate-parentheses/)

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

## Constraints:

 - 1 <= n <= 8

## Example 1:
```sh
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```

## Example 2:
```sh
Input: n = 1
Output: ["()"]
```
